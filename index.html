2. Dati raccolti e finalità

MushScout raccoglie i seguenti dati personali:

📧 Dati di account
  • Indirizzo email — per registrazione e autenticazione
  • Nome e cognome / username — per il profilo utente
  • Foto profilo (opzionale) — caricata dall'utente
📍 Dati di localizzazione
  • Coordinate GPS precise — salvate per ogni ritrovamento di funghi
  • Percorso GPS — tracciato durante le sessioni di raccolta
  • Posizione in background — durante una sessione di raccolta avviata dall'utente, l'app registra il tracciato GPS anche a schermo spento o in secondo piano, tramite un foreground service con notifica persistente. Richiede il permesso di posizione "Sempre" su Android. Il tracciamento si attiva solo durante una sessione attiva
  • La localizzazione è utilizzata solo per le funzionalità principali dell'app; le posizioni dei ritrovamenti sono private e non vengono condivise con altri utenti né con terzi a fini pubblicitari
📸 Foto e contenuti
  • Foto dei funghi — caricate volontariamente dall'utente, salvate su Cloudflare R2 (data center in Europa). Le foto scattate offline restano sul dispositivo (IndexedDB) finché non torna la connessione
  • Le foto vengono compresse prima del caricamento
  • Le foto non sono mai pubbliche di default
📢 Pubblicità (solo utenti free)
  • Per gli utenti della versione gratuita, Google AdMob può raccogliere l'identificatore pubblicitario di Android (Advertising ID) e dati di utilizzo per mostrare e misurare gli annunci
  • Puoi reimpostare o eliminare l'Advertising ID in Impostazioni → Google → Annunci. Gli utenti Premium non vedono pubblicità
📊 Dati di utilizzo
  • Numero di ritrovamenti, sessioni, achievement
  • Statistiche di utilizzo aggregate (non identificative)
  • Token dispositivo per notifiche push (Firebase Cloud Messaging)

3. Base giuridica del trattamento

  • Esecuzione del contratto (art. 6.1.b GDPR) — per fornire le funzionalità dell'app
  • Consenso (art. 6.1.a GDPR) — per notifiche push e pubblicità personalizzata
  • Interesse legittimo (art. 6.1.f GDPR) — per sicurezza e prevenzione abusi

2. Data Collected and Purposes

MushScout collects the following personal data:

📧 Account Data
  • Email address — for registration and authentication
  • Name / username — for the user profile
  • Profile photo (optional) — uploaded by the user
📍 Location Data
  • Precise GPS coordinates — saved for each mushroom finding
  • GPS route — tracked during foraging sessions
  • Background location — during a user-initiated foraging session, the app records the GPS track even with the screen off or in the background, via a foreground service with a persistent notification. This requires "Always" location permission on Android. Tracking is active only during an active session
  • Location is used only for core app features; find locations are private and never shared with other users or with third parties for advertising purposes
📸 Photos and Content
  • Mushroom photos — voluntarily uploaded by the user, stored on Cloudflare R2 (EU data centers). Photos taken offline stay on the device (IndexedDB) until the connection returns
  • Photos are compressed before upload
  • Photos are never public by default
📢 Advertising (free users only)
  • For free-version users, Google AdMob may collect the Android Advertising ID and usage data to serve and measure ads
  • You can reset or delete the Advertising ID in Settings → Google → Ads. Premium users see no ads
📊 Usage Data
  • Number of finds, sessions, achievements
  • Aggregate usage statistics (non-identifying)
  • Device token for push notifications (Firebase Cloud Messaging)
